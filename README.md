# ForumHub — Веб-платформа для обсуждений и форумов

ForumHub — это современная веб-платформа для обсуждений, вдохновлённая Reddit. Проект разработан в рамках курсовой работы по специальности «Программное обеспечение».

## 📋 Функционал

- ✅ Регистрация и авторизация пользователей (JWT)
- ✅ Просмотр категорий и тем
- ✅ Создание новых тем (постов)
- ✅ Комментарии с возможностью ответов (иерархические)
- ✅ Голосование за комментарии (лайки/дизлайки)
- ✅ Личный кабинет с профилем
- ✅ Избранные темы
- ✅ Поиск по платформе
- ✅ Пагинация (кнопка «Загрузить ещё»)
- ✅ Тёмная/светлая тема оформления

## 🛠 Технологический стек

| Компонент | Технология |
|-----------|------------|
| Frontend | React.js |
| Маршрутизация | React Router |
| HTTP-запросы | Axios |
| Backend | Node.js + Express |
| База данных | MongoDB |
| Аутентификация | JWT |

## 🚀 Установка и запуск

### Требования

- Node.js (версия 16 или выше)
- MongoDB (локально или облако)

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш_username/forumhub.git
cd forumhub
2. Настройка бэкенда
bash
cd server
npm install
Создайте файл .env в папке server:

text
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/forum_db
JWT_SECRET=your_jwt_secret_key
Запустите сервер:

bash
node server.js
Сервер запустится на http://localhost:5000

3. Настройка фронтенда
bash
cd ../forum-client
npm install
npm start
Фронтенд запустится на http://localhost:3000

4. Регистрация и вход
Откройте http://localhost:3000

Нажмите «Регистрация» и создайте аккаунт

Войдите в систему

Начинайте создавать темы и комментарии!

## Куда сохранить файл?

### 1. В GitHub (онлайн)

1. Зайди в свой репозиторий на GitHub
2. Нажми **«Add file»** → **«Create new file»**
3. Назови файл **`README.md`**
4. Скопируй текст выше и вставь
5. Нажми **«Commit changes»**

### 2. В проекте (локально)

1. Открой папку `C:\Users\Хамид\My-Forum\`
2. Создай новый текстовый файл
3. Переименуй его в **`README.md`** (важно расширение `.md`, а не `.txt`)
4. Открой блокнотом, вставь текст, сохрани
